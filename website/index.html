Профессиональная проверка ПК на читы

Элитная forensic-утилита для проверки компьютера, разработанная для моментального обнаружения скрытых модификаций, чит-клиентов, сторонних библиотек, логов и следов в реестре.

01

Журнал изменений USN NTFS

Отслеживает удаленные перед проверкой читы (.exe, .dll, .jar, .pf). Даже если игрок очистил корзину, NTFS логи сохраняют точную историю и время удаления.

02

Анализ памяти JVM & DLL

Сканирует активный стек javaw.exe на предмет несанкционированных динамических DLL-модулей, загруженных из временных папок (таких как vec.dll или InjGen).

03

Сверка хешей Modrinth API

Вычисляет SHA-1/SHA-256 хеши установленных модов и проверяет их через официальную базу Modrinth API для быстрого выявления вшитого вредоносного чит-кода.

04

Продвинутый Prefetch парсер

Декодирует Prefetch (.pf) файлы Windows для извлечения количества запусков софта, точных размеров бинарников, дат запусков и уникальных системных хешей.

05

Поиск в архивах логов

Автоматически распаковывает все архивные файлы .log.gz в папке логов игры для детального поиска следов команд читов, биндов и запрещенных подключений.

06

Инспектор DNS & Netstat

Опрашивает системный кэш DNS клиента и установленные сетевые сокеты для обнаружения IP/доменов авторизации приватных читов и вебхуков Discord/Telegram.

Современный и понятный интерфейс

Удобная интерактивная диагностическая панель с отображением логов активности, списков удаленного софта и вердиктов проверок.

Гарантия безопасного самоуничтожения

AuroraChecker содержит встроенный, безопасный скрипт очистки. Он точечно удаляет только сам исполняемый файл, скачанные утилиты и PID процесса. Ваши личные файлы, папка «Загрузки» и файлы на Рабочем столе полностью защищены.

Цифровая подпись UAC

Файл подписан сертификатом CN=zentixctm с валидным статусом Authenticode.

Запуск без установки

Работает мгновенно через pywebview, не оставляет мусора в реестре Windows.

Очистка в один клик

Абсолютно никаких логов и файлов на проверяемом ПК после завершения сессии.
