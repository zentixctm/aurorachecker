01

USN NTFS Journal Scan

Tracks deleted modified cheat files (.exe, .dll, .jar, .pf) in real-time. Even if the player deletes the files before the checking session, NTFS logs hold the history.

02

JVM RAM Injections

Scans the active javaw.exe memory stack to locate unauthorized dynamic DLL modules loaded from user temp folders (like vec.dll or InjGen hookers).

03

Modrinth Hash Check

Calculates SHA-1/SHA-256 hashes of all installed mods and performs a batch verification query through official Modrinth API endpoints to detect backdoored releases.

04

Advanced Prefetch parser

Parses Prefetch (.pf) files to read exact execution run counters, binary sizes, last launch timestamps and hashes of executed apps.

05

Minecraft Log Archives

Automatically decompresses all .log.gz files inside the logs folder to parse ancient cheat command calls, bindings, and unauthorized connections.

06

DNS & Netstat Inspector

Queries Windows DNS cache and established netstat sockets to highlight cheat authorization portals, Discord/Telegram webhooks and filesharing domains.
